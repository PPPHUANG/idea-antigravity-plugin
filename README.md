# Idea Antigravity Plugin

A simple IntelliJ IDEA plugin that allows you to open current file in Antigravity.

## Installation

1. Clone the repository.
2. Build the plugin using `./gradlew buildPlugin`.
3. Install the generated ZIP file from `build/distributions/` via IntelliJ's "Install Plugin from Disk" feature.

## Usage

Right-click on any file in the Project View or Editor -> "Open in Antigravity".
